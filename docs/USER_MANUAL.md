## 快速開始

### 第一次使用

1. **啟動程式**
   - Windows: 雙擊 `start.bat` 或執行 `python gui_main.py`
   - macOS/Linux: 執行 `./start.sh` 或 `python3 gui_main.py`

2. **初始設定**
   - 程式首次啟動會開啟設定視窗
   - 輸入 Google Gemini API 金鑰（必要）
   - 選擇預設工作目錄
   - 設定輸出路徑

3. **測試功能**
   - 準備一個小的音訊檔案（建議 < 5MB）
   - 使用語音轉錄功能進行測試
   - 確認輸出結果正確

### 基本工作流程

```
音訊檔案 → 語音轉錄 → AI 分析 → 媒體歸檔 → 智能搜尋
```

## 功能詳解

### 語音轉錄

#### 支援格式
- **輸入格式**: MP3, WAV, M4A, FLAC, OGG, WMA
- **輸出格式**: TXT, SRT, VTT, JSON

#### 單檔案轉錄

1. **選擇檔案**
   - 點擊「選擇音訊檔案」按鈕
   - 或拖拽檔案到程式視窗

2. **設定參數**
   - 選擇輸出格式
   - 設定語言（自動檢測或手動選擇）
   - 選擇輸出路徑

3. **開始轉錄**
   - 點擊「開始轉錄」按鈕
   - 觀察進度條和狀態訊息
   - 轉錄完成後可預覽結果

#### 批次轉錄

1. **選擇資料夾**
   - 點擊「批次轉錄」標籤
   - 選擇包含音訊檔案的資料夾

2. **設定批次選項**
   - 選擇要處理的檔案類型
   - 設定輸出格式（可多選）
   - 選擇輸出目錄結構

3. **執行批次處理**
   - 點擊「開始批次轉錄」
   - 監控整體進度
   - 查看處理報告

#### 進階選項

- **AI 校正**: 使用 AI 自動校正轉錄結果
- **說話者識別**: 識別不同說話者（實驗性功能）
- **時間戳記**: 在輸出中包含精確時間戳記
- **信心度過濾**: 過濾低信心度的轉錄結果

### AI 分析

#### 分析類型

1. **內容摘要**
   - 自動生成文字摘要
   - 提取關鍵資訊
   - 識別重要段落

2. **標籤生成**
   - 自動生成相關標籤
   - 分類內容主題
   - 識別情感傾向

3. **關鍵字提取**
   - 提取重要關鍵字
   - 計算詞頻統計
   - 識別專業術語

4. **結構分析**
   - 分析文字結構
   - 識別章節段落
   - 提取標題層次

#### 使用步驟

1. **選擇檔案**
   - 從檔案清單中選擇要分析的文字檔案
   - 支援 TXT, SRT, VTT 格式

2. **選擇分析類型**
   - 勾選需要的分析功能
   - 設定分析參數

3. **執行分析**
   - 點擊「開始分析」
   - 等待 AI 處理完成
   - 查看分析結果

4. **儲存結果**
   - 將分析結果儲存為 JSON 或 TXT
   - 匯出分析報告

### 智能搜尋

#### 搜尋類型

1. **關鍵字搜尋**
   ```
   會議記錄
   ```

2. **片語搜尋**
   ```
   "專案進度討論"
   ```

3. **布林搜尋**
   ```
   會議 AND 記錄 NOT 草稿
   ```

4. **自然語言搜尋**
   ```
   找出上個月關於預算的會議記錄
   ```

#### 進階搜尋語法

- **日期範圍**: `date:2024-01-01..2024-01-31`
- **檔案類型**: `type:srt` 或 `ext:txt`
- **檔案大小**: `size:>1MB`
- **標籤過濾**: `tag:重要`
- **路徑過濾**: `path:會議記錄/`

#### 搜尋範本

建立和使用搜尋範本：

1. **建立範本**
   - 輸入複雜搜尋查詢
   - 點擊「儲存為範本」
   - 輸入範本名稱和描述

2. **使用範本**
   - 從範本清單中選擇
   - 修改參數（如需要）
   - 執行搜尋

3. **管理範本**
   - 編輯現有範本
   - 刪除不需要的範本
   - 匯入/匯出範本

### 媒體歸檔

#### 歸檔規則

1. **按日期歸檔**
   ```
   年份/月份/檔案名稱
   例: 2024/01/會議記錄_20240115.txt
   ```

2. **按類型歸檔**
   ```
   檔案類型/子分類/檔案名稱
   例: 音訊/會議記錄/weekly_meeting.mp3
   ```

3. **按標籤歸檔**
   ```
   標籤/日期/檔案名稱
   例: 重要/2024-01/project_discussion.srt
   ```

4. **自訂規則**
   - 使用正規表達式
   - 基於檔案內容
   - 結合多種條件

#### 歸檔流程

1. **選擇來源**
   - 選擇要歸檔的檔案或資料夾
   - 設定過濾條件

2. **設定規則**
   - 選擇歸檔規則
   - 預覽目標結構
   - 設定命名規則

3. **執行歸檔**
   - 確認歸檔計劃
   - 開始歸檔處理
   - 查看歸檔報告

## 故障排除

### 常見問題

#### 1. 程式無法啟動

**症狀**: 雙擊啟動檔案後程式沒有反應

**可能原因**:
- Python 環境問題
- 依賴套件缺失
- 權限不足

**解決方案**:
1. 檢查 Python 版本（需要 3.8+）
2. 重新安裝依賴套件：`pip install -r requirements.txt`
3. 以管理員權限執行
4. 查看錯誤日誌：檢查 `logs/error.log`

#### 2. 轉錄功能異常

**症狀**: 音訊檔案無法轉錄或轉錄結果為空

**可能原因**:
- 音訊檔案格式不支援
- 檔案損壞
- 磁碟空間不足
- 網路連線問題

**解決方案**:
1. 確認檔案格式在支援清單中
2. 嘗試用其他播放器播放檔案
3. 檢查可用磁碟空間
4. 測試網路連線
5. 重新啟動程式

#### 3. AI 分析失敗

**症狀**: AI 分析功能報錯或無回應

**可能原因**:
- API 金鑰無效或過期
- API 配額用盡
- 網路連線問題
- 檔案內容過長

**解決方案**:
1. 檢查 API 金鑰設定
2. 確認 API 配額狀態
3. 測試網路連線
4. 分割大檔案後重試
5. 查看 API 使用日誌

#### 4. 搜尋結果不準確

**症狀**: 搜尋無法找到預期的檔案

**可能原因**:
- 搜尋索引過期
- 檔案路徑變更
- 搜尋語法錯誤
- 檔案權限問題

**解決方案**:
1. 重建搜尋索引
2. 更新檔案路徑
3. 檢查搜尋語法
4. 確認檔案權限
5. 清除搜尋快取

```

├── 原始檔案/
│   ├── 音訊/

│   └── 文件/
```
